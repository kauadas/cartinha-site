# Pasta de Sprites

Coloque seus arquivos PNG de sprites nesta pasta com os seguintes nomes:

- `peixe-dourado.png` - Sprite do peixe dourado
- `peixe-azul.png` - Sprite do peixe azul
- `tubarao-baleia.png` - Sprite do tubarão baleia
- `tartaruga.png` - Sprite da tartaruga marinha
- `polvo.png` - Sprite do polvo

## Recomendações de tamanho:

- Peixe Dourado: ~50x35px
- Peixe Azul: ~50x35px
- Tubarão Baleia: ~80x45px
- Tartaruga: ~55x45px
- Polvo: ~50x60px

Os sprites serão ajustados automaticamente pelo CSS para se adaptar ao tamanho da tela.
